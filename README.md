## 🌟 Key Features

| Feature | Description |
| :--- | :--- |
| **🛡️ Anti-Delete** | Instantly restores messages deleted by others. |
| **📞 Anti-Call** | Automatically rejects and blocks incoming calls. |
| **🔗 Anti-Link** | Auto-kicks members who share group links (customizable). |
| **⏳ OpenTime** | Schedule group openings with auto-close timers. |
| **🎭 Reactions** | Huge library of Anime-style GIFs for interactions. |
| **🔋 Auto-Bio** | Real-time status updates in WhatsApp About (Uptime/RAM). |
| **📱 Pairing Code** | Link with phone number directly (No QR scan needed). |

## 🚀 Quick Setup

### 1. Requirements
* [Node.js](https://nodejs.org/) (v16 or higher)
* [Git](https://git-scm.com/)
* A secondary WhatsApp account (for safety)

### 2. Installation
```bash
# Clone the repository
git clone [https://github.com/Neggy5/OPTIMUS-PRIME.git](https://github.com/Neggy5/OPTIMUS-PRIME.git)

# Enter the directory
cd OPTIMUS-PRIME

# Install dependencies
npm install
